# Contributing to Dara

Thank you for your interest in contributing to Dara.

Dara is a digital Ayo strategy game exploring cultural gameplay, modern web interfaces, and future Stellar/Soroban-powered game infrastructure.

## How to Contribute

1. Check the Issues tab.
2. Choose an issue that matches your skill level.
3. Comment on the issue before starting.
4. Fork the repository.
5. Create a branch for your work.
6. Make focused changes.
7. Open a pull request with a clear description.

## Pull Request Checklist

Before opening a pull request, please confirm:

- Your change matches the issue scope.
- Documentation is updated where needed.
- The PR description explains what changed.
- The PR does not include unrelated changes.
- No secrets, private keys, or local environment files are committed.

## Issue Complexity

- Trivial = small documentation or setup task
- Medium = moderate feature, UI, testing, or planning task
- High = research-heavy, architecture, or complex implementation task
