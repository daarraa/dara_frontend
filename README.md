## Overview

Dara is a digital Ayo strategy game project inspired by traditional African board gaming culture.

This repository is planned to contain the user-facing game interface, including the Ayo board UI, player experience, game status screens, wallet-ready identity flows, and future Stellar/Soroban-powered gameplay features.

## What is Dara?

Dara brings the traditional Ayo game experience into a modern digital interface. The goal is to preserve cultural strategy gameplay while exploring how Stellar and Soroban can support player identity, match records, rewards, and tournament-style play.

## Planned Features

- Responsive Ayo board interface
- Player turn display
- Seed and pit representation
- Local gameplay state management
- Mobile-friendly gameplay experience
- Stellar wallet connection research
- Future Soroban match and reward integration

## Stellar/Soroban Roadmap

Dara is preparing Stellar ecosystem support through:

- Stellar wallet-based player identity
- Soroban-powered match records
- Game result verification
- Tournament and rewards research
- Transparent gameplay contribution history
