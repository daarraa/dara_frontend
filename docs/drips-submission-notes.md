# Dara Drips Submission Notes

## Short Project Description

Dara is a digital Ayo strategy game inspired by traditional African board gaming culture, exploring modern web gameplay, Stellar wallet identity, Soroban-powered match records, and future game rewards.

## Repositories

- dara_frontend: planned user-facing game interface, Ayo board UI, player experience, and wallet-ready frontend work.
- dara_contract: contract architecture, Soroban research, match records, rewards, and future smart contract implementation.

## Backend Note

The backend repository is not included in this submission because the current Wave work is focused on the frontend and contract architecture layers.

## Types of Issues

We will post scoped issues covering documentation, game UI, accessibility, Stellar wallet research, Soroban match architecture, game-state planning, test planning, and rewards/tournament flow research.

## Repository Relationship

The selected repositories are part of the same Dara project. `dara_frontend` contains the planned user-facing game interface, while `dara_contract` contains the planned contract architecture and Stellar/Soroban research for match records, rewards, and on-chain game infrastructure.
